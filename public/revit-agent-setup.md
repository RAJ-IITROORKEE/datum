# Datum Revit Agent Setup

1. Open Copilot and generate a pairing code from the Revit menu in the header.
2. Download and run `DatumRevitAgent.exe` from:

```text
/downloads/DatumRevitAgent.exe
```

on the same machine as Revit.
3. Paste the pairing code when prompted.
4. Keep Revit open and plugin switch ON.
5. Run commands from chat using `/run <tool_name> <json_args>`.

Example:

```text
/run get_levels_list {}
```
